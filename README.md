Neuro Notes Website

Official Product Website for Neuro Notes
Built with React, Vite, and Tailwind CSS, this repository hosts the Neuro Notes landing page, showcasing the AI-powered desktop notes application and providing download links for users.

Overview
This repository contains the full source code for the Neuro Notes product website, designed to:
Present Neuro Notes as a real-world software product.
Highlight app features, AI capabilities, and technical stack.
Allow users to download the app directly.
Improve your portfolio visibility for potential employers and collaborators.

Features

Responsive Landing Page: Works on desktop and mobile devices.
Hero Section & CTA: Highlights key app features and download options.
Sections: Features, Use Cases, Screenshots, Tech Stack, Model Info, FAQ, Settings, Contact.
Product Download: Provides direct links to the Neuro Notes desktop installer.
Modern UI/UX: Built with React + Vite, styled for a professional product look.
Portfolio Ready: Demonstrates real-world software presentation.

Tech Stack

Frontend: React, Vite, TypeScript
Styling: Tailwind CSS
Deployment: Vercel
Assets: Figma for design mockups, optimized images
Deployment: GitHub Pages / Netlify / Vercel (depending on your hosting)
